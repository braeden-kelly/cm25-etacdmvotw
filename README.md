# Equation to Animation: Crafting Dynamic Math Visuals on the Web

**Presenter:** Courtney Yatteau  
**Event:** CodeMash 2025  
**Time:** Wednesday, January 15, 1:00 PM - 5:00 PM  
**Room:** Salon E  

Explore how to make math engaging and accessible through dynamic data visualizations. In this workshop, you'll learn to create interactive graphs, charts, and visual tools using HTML, CSS, and JavaScript, leveraging powerful libraries like D3.js and Chart.js.

## Workshop Goals
- Understand the basics of data visualization.
- Master the use of D3.js and Chart.js for interactive visuals.
- Integrate geospatial elements into your visualizations.
- Apply learned concepts to real-world projects.

## Agenda
1. Goals & Value of Data Visualization  
2. Tools & Setup (VS Code, D3.js, Chart.js)  
3. Creating Customizable Charts and Graphs  
4. Advanced Visualizations with D3.js  
5. Wrap-Up and Q&A
